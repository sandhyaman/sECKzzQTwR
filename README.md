# 项目简介

## 核心功能
- 数据可视化大屏
- 数据分析
- 登陆注册
- 用户管理
- 公告管理
- 版本管理
- 数据管理

## 适用场景
- 公安机关警情分析
- 政府部门安全监控
- 企业安全风险评估

## 技术特点
- 使用Python和Flask构建，提供灵活的扩展性
- 基于Pyecharts实现数据可视化，支持多种图表类型
- 源码和SQL文件提供，方便二次开发和数据库导入
- 可运行环境配置指南详细，易于上手
